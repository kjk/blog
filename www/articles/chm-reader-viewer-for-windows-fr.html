Liste de lecteurs et visionneuses CHM pour Windows

lecteurs et visionneuses CHM

Sumatra PDF

Sumatra PDF supporte l'affichage des fichiers CHM depuis sa version 1.9. Sumatra is gratuit, open-source and supporte plusieurs autres formats, tels que PDF, ePub, MOBI, XPS, DjVu, CBR, CBZ.

XCHM

XCHM est une visionneuse CHM multi-plateforme et open-source. Elle fonctionne sous Windows, Mac et Linux.

FBReader

FBReader est un lecteur de e-Books multi-plateforme et open-source. Il peut aussi lire des fichiers CHM.

CHM Reader extension for FireFox

CHM Reader n'est pas un programme autonome mais une extension (un module) FireFox qui permet de lire les fichiers CHM directement dans le navigateur.

ExtraCHM

ExtraCHM est un lecteur CHM commercial.

Beyond CHM

Beyond CHM est un lecteur et éditeur CHM commercial avec onglets.

UltraCHM

UltraCHM est un lecteur CHM commercial.

Mobipocket Reader

Mobipocket Reader supporte, parmi plusieurs autres, le format CHM.

Calibre

Calibre est un logiciel de gestion de bibliothèques avec une visionneuse intégrée permettant de lire / convertir de nombreux formats de fichiers, principalement ceux utilisés pour les e-Books.

Renseignements sur le format CHM

CHM est un acronyme pour les fichiers Microsoft Compressed HTML, un format propriétaire de Microsoft pour fournir un système d'aide et de documentation pour les applications, en remplacement de l'ancien format Microsoft WinHelp.

Un document CHM est un ensemble de pages HTML, compressés et compactés dans un fichier unique. Ils sont compressés à l'aide de l'algorithme LZX. Un document CHM contient aussi des fichiers complémentaires, comme un index de recherche, une table des matières, etc. Pour une description plus complète, du format CHM, vous pouvez consulter la page Unofficial HTML Help Specification (Spécifications non officielles de l'aide HTML, en anglais).
