Histórico das versões

1.6 (2011-05-30)

Alterações nesta versão

  • adicionado o suporte a documentos DjVu
  • adicionado um ecrã de Documentos recentes se nnão existir qualquer documento aberto
  • adicionado o suporte a documentos Postscript (requer que estaja instalada uma versão recente do Ghostscript)
  • adicionado o suporte a exibir uma pasta que contenha imagens: basta arrastar a pasta para uma janela do SumatraPDF
  • adicionado o suporte para abrir ligações existentes nos documentos
  • adicionado o suporte às tabelas de conteúdos de documentos XPS
  • adicionado o ecrã de progresso das impressões com possibilidade de as cancelar
  • adicionado um botão para Imprimir na barra de ferramentas
  • experimental: pré-visualização dos documentos PDF nos Windows Vista e 7. Criação de miniaturas e exibe-as no painel de pré-visualizações do Explorador do Windons. Necessita de especificar a opção durante a instalação. É possível que não funcione em Windows 7 x64.
1.5.1 (2011-04-26)

Alterações nesta versão:

  • corrigidos alguns erros de exibição
1.5 (2011-04-23)

Alterações nesta versão:

  • adicionado o suporte a documentos XPS
  • adicionado o suporte a livros comic CBZ e CBR
  • adicionado um item de menu em Ficheiro/Gravar atalho para criar atalhos diretos a um local específico do documento
  • adicionado um menu de contexto para copiar texto, ligar a endereços e comentários. No "plug-in" do navegador web também permite os comandos Gravar e Imprimir
  • adicionado o suporte a navegação nas pastas (Ctrl+Shift+Seta direita abre o documento seguinte da pasta, Ctrl+Shift+Seta esquerda abre o documento anterior)
1.4 (2011-03-12)

Alterações nesta versão:

  • "plug-in" para ver documentos no(s) Firefox/Chrome/Opera (não existe suporte para ao Internet Explorer). Por omissão, o "plug-in" não está selecionado para instalação. O utilizador tem que escolher a opção durante a instalação
  • IFilter para permitir a pesquisa de documentos PDF através do Windows Desktop Search (i.e. pesquisa a partir do menu Iniciar nos Windows Vista/7). Também não é instalado por omissão
  • adicionada a possibilidade de deslocar documento com o botão direito do rato
  • adicionada a possibilidade de escolher o diretório para o programa durante a instalação
  • adicionados itens de menu para abrir os documentos no Foxit ou no PDF-XChange (caso estejam instalados)
  • o instalador já não é comprimido com mpress. Esta compressão fazia com que alguns programas anti-vírus identificassem o Sumatra como um vírus. Só que agora os executáveis também são maiores. Nota: a versão .zip ainda é comprimida
  • foi removida a opção de linha de comandos -title
  • adicionado o suporte para documentos encriptados em AES-256
  • corrigido um erro de "integer overflow" reportado por Jeroen van der Gun e outras pequenas correções/melhorias ao nível da manipulação de documentos
1.3 (2011-02-04)

Alterações nesta versão:

  • melhorias na seleção e cópia de textos. Agora replicamos o método como o navegador ou o Adobe Reader funcionam: basta selecionar o texto com o rato e utilizar Ctrl-C para o copiar para a área de transferência
  • Shift+Botão esquerdo do rato para deslocar o documento, Ctrl+Botão esquerdo do rato para uma seleção retangular (para a cópia de imagens)
  • tecla "c" troca o para o modo contínuo
  • "+" / "*" no teclado numérico para ampliar e rodar o documento
  • adicionados à barra de ferramentas os ícones "Ajustar à página" e "Ajustar à largura"; atualização do aspeto dos ícones da barra de ferramentas
  • adicionado o suporte para utilizar os botões do rato voltar/avançar para uma melhor navegação
  • a versão 1.2 introduziu um novo modo de ecrã completo que ficou como omissão. O método antigo ainda estava disponível, mas não era de fácil utilização. Adicionámos o item de menu "Ver/Apresentação" para o novo modo de ecrã completo bem como para o antigo tornando-o assim mais fácil de utilizar
  • novo e melhorado instalador
  • melhorias ao nível da ampliação (agora já é possivel ampliar até 6400%)
  • a pesquisa de texto utiliza menos memória
  • melhorias no sistema de impressão
  • atualização de traduções
  • atualização da versão do mupdf com correções de erros e algumas melhorias
  • utilização da biblioteca libjpeg-turbo em vez de libjpeg, para uma descodificação mais rápida de alguns PDFs
  • atualização da biblioteca openjpeg para a versão 1.4 e atualização do freetype para a versão 2.4.4
  • corrigidos 2 erros de "overflow" reportados por Stefan Cornelius da Secunia Research

Contributos nesta versão:
