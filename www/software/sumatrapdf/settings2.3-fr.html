Personnalisation de SumatraPDF

On peut changer l'aspect et le comportement de SumatraPDF en éditant le fichier SumatraPDF-settings.txt. Le fichier est sauvegardé dans le répertoire %APPDATA%\SumatraPDF pour la version installée ou dans le même répertoire que l'exécutable SumatraPDF.exe executable pour la version portable.

Utiliser l'option de menu Préférences -> Options avancées... pour ouvrir le fichier de configuration avec l'éditeur de texte par défaut.

Le fichier est un simple fichier texte. Ci-dessous, une explication de la signification des différents paramètres, et leur valeur par défaut.

Les paramètres en surbrillance ne peuvent être changés dans l'interface utilisateur. La modification des autres paramètres directement dans ce fichier n'est pas recommandé.

Pour ajouter ou supprimer des lignes comportant des crochets droits, bien s'assurer de toujours ajouter ou supprimer les crochets droits par paires ! Faute de quoi, on risque de perdre toutes le données qui les suivent.

Syntaxe pour les valeurs des  couleurs

La syntaxe pour les couleurs est : #rrggbb.

Les composantes sont des valeurs hexadécimales (dans la plage 00 ... FF), qui représentent :

  • rr : la composante rouge
  • gg : la composante verte
  • bb : la composante bleue
Par exemple, #ff0000 signifie : « couleur rouge ». Vous pouvez utiliser Color Picker or ColorScheme Designer pour choisir une couleur.
