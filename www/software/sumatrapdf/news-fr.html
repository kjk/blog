Sumatra PDF est un lecteur PDF, ePub, MOBI, CHM, XPS, DjVu, CBZ, CBR pour Windows

Historique

2.4 (10-05-2013)

Changements dans cette version :

  • mode plein écran pour les livres numériques (Ctrl-L)
  • nouveaux raccourcis-clavier :
    • F9 – afficher/masquer la barre de menus (ne s'enregistre pas en fermant)
    • F8 – afficher/masquer la barre d'outils
  • support du format d'images WebP  (standalone and in comic books)
  • support des bandes dessinées compressées au format RAR5
  • correction de multiples plantages
2.3.2 (25-05-2013)

Changement dans cette version :

  • correction de bogue dans le changement de langue via le menu Préférences/Changement de langue
2.3.1 (23-05-2013)

Changement dans cette version :

  • ne requiert pas SSE2 (pour le support des ordinateurs anciens, sans instructions SSE2)
2.3 (22-05-2013)

Changements dans cette version :

  • plus grande configurabilité via les options avancées
  • "Aller à la page" dans l'interface-utilisateur des livres numériques
  • ajout du menu Affichage/Mode manga pour les bandes dessinées (fichiers CBZ/CBR)
  • nouveaux raccourcis-clavier :
    • Ctrl-Up : page précédente
    • Ctrl-Down : page suivante
  • ajout du support des documents OpenXPS
  • support de l'algorithme Deflate64 pour les fichiers Comic Book (CBZ/CBR)
  • correction : retrait de paragraphe manquant dans les documents EPUB
  • l'impression avec l'option "Utiliser les dimensions d'origine" ne centre plus les pages sur la feuille
  • taille réduite. L'installeur est environ 1 Mo plus petit
  • inconvénient : cette version ne supporte plus les processeurs très anciens, sans instructions SSE2. L'utilisation de SSE2 rend Sumatra plus rapide. Si vous avez un vieil ordinateur sans SSE2, il vous faudra utiliser la version 2.2.1.
2.2.1 (12-01-2013)

Changements dans cette version :

  • correction : un livre numérique parfois ne se rappelait pas la position de lecture antérieure
  • correction : Sumatra ne se fermait plus après ouverture de fichiers sur un lecteur réseau
  • corrections des plantages les plus fréquents et de la robustesse de l'analyseur PDF
2.2 (24-12-2012)

Changements dans cette version :

  • ajout du support du format de livre numérique FictionBook
  • ajout du support des documents PDF chiffrés avec Acrobat X
  • option de compatibilité “Imprimer en tant qu'image” dans le dialogue d'impression pour un affichage correct des mangas
  • nombreuses petites améliorations et corrections de robustesse
2.1.1 (07-05-2012)

Changements dans cette version:

  • correction de quelques bogues
2.1 (03-05-2012)

Changements dans cette version:

  • support du format de livre numérique EPUB
  • ajout d'un menu Ficher/Renommer pour renommer le fichier en cours (contribution de Vasily Fomin)
  • support des fichiers TIFF multi-pages
  • support des images TGA
  • support pour plusieurs métadonnées des fichiers CBZ (bandes dessinées)
  • support pour les images JPEG XR (disponible dans Windows Vista et supérieur, pour Windows XP le Windows Imaging Component doit être installé)
  • l'installeur est maintenant signé

1.7 (18-07-2011)

Changements dans cette version:

  • favoris
  • support amélioré des langues « de droite à gauche », tel l'arabe
  • les numéros de pages « logiques » (c.-à-d., non automatiques, comme par exemple i, ii, iii, etc.) sont affichés et utilisés, si un document les fournit
  • permet de restreindre les fonctionnalités de SumatraPDF avec plus de granularité. Voir sumatrapdfrestric.init pour plus d'informations
  • -named-dest recherche aussi les chaînes de caractères dans la table des matières
  • support amélioré des fichiers EPS (nécessite Ghostscript)
  • installeur plus fiable et plus efficace
  • nombreuses améliorations et corrections de bogues mineurs
1.6 (30-05-2011)

  • ajout du support pour l'affichage de répertoires contenant des images glissés dans la fenêtre SumatraPDF
  • support des liens cliquables : table des matières pour les documents XPS
  • affiche la progression de l'impression et permet son annulation
  • ajout d'un bouton d'impression dans la barre d'outils
  • expérimental : prévisualisation de documents PDF dans Windows Vista and 7. Crée des miniatures et affiche les documents dans la prévisualisation du panneau d'exploration. Doit être explicitement sélectionné durant l'installation. Des comptes-rendus indiquent que ça ne fonctionne pas avec Windows 7 x64.

1.4 (12-03-2011)

Changements dans cette version:

  • plugin pour les navigateurs internet Firefox/Chrome/Opera (Internet Explorer non supporté). Par défaut, non installé.
  • IFilter, qui permet la recherche de texte dans les fichiers PDF pour la fonction de recherche intégrée de Windows (Windows Desktop Search, accessible dans le menu démarrer ou le menu contextuel de l'Explorateur Windows). N'est pas non plus installé par défaut
  • défilement avec le bouton droit de la souris
  • possibilité de choisir le répertoire d'installation avec l'installeur
  • options de menu pour rouvrir le document courant dans Foxit et PDF-XChange (s'ils sont installés)
  • nous ne compressons plus l'exécutable dans l'installeur avec mpress. Il produit une identification erronée de virus dans Sumatra par certains programmes anti-virus. L'inconvénient est que le programme occupe plus d'espace disque, désormais. Note: nous compressons toujours la version portable dans le .zip
  • suppression du paramètre de ligne de commandes -title
  • support pour les fichiers PDF chiffrés avec AES-256
  • correction d'un dépassement d'entier signalé par Jeroen van der Gun et quelques autres corrections mineures et améliorations pour la prise en charge des PDF
1.3 (04-02-2011)

Changements dans cette version:

  • sélection et copie de texte améliorée. Nous imitons désormais la méthode dont un navigateur internet ou Adobe Reader travaillent : simplement sélectionner le texte avec la souris et utiliser Ctrl-C pour le copier dans le presse-papier
  • Maj + Bouton gauche de la souris permet maintenant de faire défiler le document, Ctrl + Bouton gauche crée toujours une sélection rectangulaire (pour copier une image)
  • le raccourci 'c' est une bascule en mode Vue continue/discontinue
  • '+' / '*' sur le pavé numérique permettent le zoom et la rotation
  • ajout d'icônes dans la barre d'outils pour “Page unique” et “Pleine largeur”, et mise à jour de l'apparence des icônes de la barre d'outils
  • support pour la molette de défilement de la souris pour le défilement avant/arrière
  • 1.2 introduit un nouveau mode plein écran qui devient le mode plein écran par défaut. L'ancien mode restait disponible mais peu aisément accessible. Nous avons ajouté une option de menu Affichage/Présentation pour le nouveau mode et Affichage/Plein écran pour l'ancien, pour rendre le premier plus visible
  • installeur nouveau et amélioré
  • performance du zoom améliorée (zoomer à 6400% ne fait plus planter le programme)
  • la recherche de texte utilise moins de mémoire vive
  • nouvelles améliorations pour l'impression
  • mise à jour des traductions
  • mise à jour avec la plus récente version de mupdf pour diverses améliorations et corrections de bogues
  • utilisation de la librairie libjpeg-turbo à la place de libjpeg, pour un décodage plus rapide de certains PDF
  • mise à jour de la librairie openjpeg avec la version 1.4 et de freetype avec la version 2.4.4
  • correction de deux problèmes de dépassement de calcul signalés par Stefan Cornelius de Secunia Research
